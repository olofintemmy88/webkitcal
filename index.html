Web Calculator:

Programming Languages: are set of grammatical rules used as instructions for computer to carry out a specific task and is a problem Oriented language used to solve a problem like the web Calculator application.

Developer tools used for the web Calculator application are:
  • HTML:is the markup language used for structuring and presenting content on the world wide web

  • CSS: Is a style sheet language used for decribing the presentation of document written in a markup language like HTML

  • JavaScript: is an Object-oriented computer programming language commonly used to create interactive effects within web browsers

  • Clients( Web browser) e.g. Google chrome, Mozilla Firefox, e.t.c

  • Http : Hypertext Transfer Protocol

  • URL(Uniform Resource Locator)

  • :http://server.com/webCalculator.html

  • Text Editor (Notepad or Notepad++) and IDE (Integrated Development environment): Is the translators that convert the html, Css and Javascript language to binary language for the computer to carry out the operation.
